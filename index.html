Sistema Binário

Cada caractere é representado por 8 bits (1 byte). Exemplo: A → 01000001

Tabela ASCII

Padrão que mapeia 128 caracteres (0-127) para números. Estendido para 256 (0-255) em ISO-8859-1.

Unicode

Padrão moderno que suporta +140k caracteres usando múltiplos bytes (UTF-8, UTF-16).

Conversões

Binário → Decimal: 01000001 = 65
Hexa: 41
Octal: 101

Como computadores processam texto: Todo texto é convertido para números binários que a CPU pode processar. Esses números são então convertidos de volta para caracteres visíveis usando tabelas de codificação.
